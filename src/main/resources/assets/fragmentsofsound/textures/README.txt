# 怪物据点纹理说明

## 需要创建的纹理文件

### 方块纹理（7个等级）
- `textures/block/monster_outpost_1.png` - 1级据点
- `textures/block/monster_outpost_2.png` - 2级据点
- `textures/block/monster_outpost_3.png` - 3级据点
- `textures/block/monster_outpost_4.png` - 4级据点
- `textures/block/monster_outpost_5.png` - 5级据点
- `textures/block/monster_outpost_6.png` - 6级据点
- `textures/block/monster_outpost_7.png` - 7级据点

### 物品纹理（7个等级）
- `textures/item/monster_outpost_1.png` - 1级据点物品
- `textures/item/monster_outpost_2.png` - 2级据点物品
- `textures/item/monster_outpost_3.png` - 3级据点物品
- `textures/item/monster_outpost_4.png` - 4级据点物品
- `textures/item/monster_outpost_5.png` - 5级据点物品
- `textures/item/monster_outpost_6.png` - 6级据点物品
- `textures/item/monster_outpost_7.png` - 7级据点物品

## 纹理规格
- 方块纹理: 16x16 或 32x32 像素
- 物品纹理: 16x16 像素
- 格式: PNG (支持透明度)

## 等级视觉建议
- **1级**: 基础外观，深色石头质感
- **2级**: 添加少量装饰元素
- **3级**: 增加更多装饰细节
- **4级**: 更复杂的装饰图案
- **5级**: 添加发光或特殊效果纹理
- **6级**: 华丽的装饰，更多细节
- **7级**: 最顶级的据点外观，带有明显的力量感

## 方块属性
- 硬度随等级递增: 3.0 → 6.0
- 所有方块都是石质材质
